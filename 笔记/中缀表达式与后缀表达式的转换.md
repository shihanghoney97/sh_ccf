### 中缀转后缀

```python
遍历表达式s:
    如果是数字:
        直接添加到后缀表达式s1
    如果是字符:
        1.栈为空，直接入栈
        2.栈不为空:
            取栈顶元素
            循环(如果栈顶元素优先级比较高or栈不为空):
                弹出栈顶元素，直至栈顶元素优先级低于字符或者栈为空
                跳出
            字符入栈
遍历栈:
    弹出栈，追加到后缀表达式s1
```

### 后缀表达式计算

```python
遍历表达式:
    如果是数字:
        入栈
    否则:
        从栈顶取出右操作数
        从栈顶取出左操作数
        计算并将结压入栈中
```
